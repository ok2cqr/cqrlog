HamLib CW setup

CW through HamLib does not need any special settings.
It uses same TCP port as your Radio one's CAT control. Only setting is default keying speed.

If you are not sure can your rig send CW via rigctld (HamLib) you may check it by opening console while cqrlog is running and typing command:
telnet localhost 4532
After getting connected type command: \dump_caps
You get several lines, one with text: 

Can send Morse:	Y

As you may guess Y means rig can send CW, N means it can not.

Commands used for HamLib CW are:

  • L KEYSPD NN (where NN is the keying speed in wpm)
  • b TEXT (TEXT is the text to be sent as CW)
